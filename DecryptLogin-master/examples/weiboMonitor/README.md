# Usage
```
usage: weiboMonitor.py [-h] [-u USERNAME] [-p PASSWORD] [-i ID]
                       [-t TIME_INTERVAL]

微博监控

optional arguments:
  -h, --help        show this help message and exit
  -u USERNAME       用户名
  -p PASSWORD       密码
  -i ID             待监控用户id
  -t TIME_INTERVAL  监控的时间间隔
```
